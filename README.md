# Stopwatch App

A simple stopwatch web application built using HTML, CSS, and JavaScript.

## Features
- Start, stop, and reset timer
- Lap recording
- Clear laps
- Clean, modern UI with animated background

## Files
- stopwatch.html – main HTML file  
- style.css – styles and layout  
- script.js – stopwatch logic and functionality  

## How to Use
1. Download or clone the project.
2. Open stopwatch.html in your browser.
3. Click START to begin timing.
4. Use LAP, STOP, and RESET as needed.

## Technologies Used
- HTML
- CSS
- JavaScript
